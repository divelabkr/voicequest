# VoiceQuest — 제품 여정 (C레벨 요약)

> 기준일 2026-06-25. 단가·시장 수치는 착수 시 재검증. 근거: `memory/voicequest-*`, 2026-06-25 3축 리서치(저비용 기술·학습앱 과금·캐릭터챗 BM, 출처 URL 보유).

## 한 줄 정체성
**"말해야 열리는 일본어 학습 RPG"** — 부끄럼 없이 입을 떼게 하고(따라하기), 목적 있는 대화로 몰입시키고(에피소드·호감도), 캐릭터와의 자유 대화를 수익으로(프리토크 구독).

핵심 해자: **목적 게이트 + 골격 판정 + 스토리 RPG(호감도·엔딩)**. 경쟁은 전부 "자유 프리토킹 도우미"형 — 이 자리는 비어 있음.

## 2. 여정 진단 — 자연스러운가?

**앞단(①→⑤)은 선형으로 매끄러움.** 가입→온보딩→따라하기(Aha)→에피소드→데일리가 온보딩·넛지·복습 버튼으로 이어짐. 엔진(음성 판정·캐시·적응 난이도·회복)은 탄탄.

**끊기는 3곳:**
1. **③→④ Aha→핵심 누수** *(2026-06-25 코드로 메움)* — 따라하기 완료 후 "다이키 만나러" 직행 CTA 추가. 첫 성공 직후 핵심 가치(서사·호감도)로 끌어올림.
2. **⑥→⑦ 수익 트리거 공백** — 프리토크(유일한 무료→유료 레버)가 미구현 → 캐시카우 BM 미가동. **최대 우선순위.**
3. **④ 핵심 콘텐츠 1개** — ep_01만 완성 → 핵심 루프 조기 소진. 리텐션 생명줄(콘텐츠 공급) 얇음.

병목은 기술이 아니라 **연결·수익 가동·콘텐츠**. 엔진은 완성, 가치가 돈으로 전환되는 마지막 두 칸이 비어 있음.

## 3. 경쟁 포지셔닝

2축: **학습 효용** × **캐릭터·서사 몰입**.

- **학습 도구**(우하): Speak($1B)·Duolingo Max·LingoDeer — 음성O, 캐릭터·서사 약함.
- **캐릭터챗**(좌상): character.ai·제타(매출 229억)·Talkie — 캐릭터 캐시카우지만 **학습 효용 0**.
- **융합**(우상): Speak·Praktika가 유니콘이나 **구독형**이고, "서사 RPG + 호감도 엔딩 + 한국어 화자용 일본어"는 **공백 = VoiceQuest 자리**.

※ 메모리의 "아오이/카나타"는 2026-06-25 실검색 미확인(이름 오류 가능) — 실재 일본 캐릭터형 = Magna·enja(650엔月)·SpeakBuddy(3,980엔月).

## 4. 비즈니스 모델 — 프리토크 캐시카우

검증: **말하기 = 고지불의향 소수 과금**(Duolingo Max 유저 8%가 매출 16%·2배가, Speak 말하기 게이트만 $1B). 캐릭터챗도 캐시카우(제타·러브앤딥 캐릭터5명 1.37조).

**3층 BM** (CLAUDE.md §0와 양립):
1. **무료** — 일일 프리토크 캡 8~10턴 + 소진 시 *차단 X → 캐시 대사 소프트 강등*(박탈감 회피).
2. **구독 ₩12,900~14,900/월** — 캡 확대(P95 공정사용, *무제한 아님*) · 음성 다양·길이 · 복습/자막 · 응답 가속.
3. **감정축 단건팩** — 에피소드·보이스팩·엔딩 분기(사전제작 변주 = §0 양립, 러브앤딥 매출 60% 근거).

❌ **버릴 것: 무제한 자유대화 종량과금** — 비용우위(캐시) 붕괴 + §0 위반 + 학습 도메인 캐시카우는 구독이 정석.

**저비용 엔진**: 임베딩 intent 게이트(`ruri`/`e5`, 자유발화도 LLM 0 다수) + rubric 등급만 Haiku 캐싱 + NPC 응답 영구 캐시 → Character.AI "$0.01/시간" 교리와 동방향. §0이 비용우위를 내장.
